//Levels
Trace        -> Detalles muy finos, solo para debugging profundo.  
Debug        -> Información técnica útil para desarrollo o pruebas.  
Information  -> Operaciones normales o exitosas del sistema.  
Warning      -> Situaciones inusuales que no detienen la ejecución.  
Error        -> Fallos en la operación actual, pero la app sigue funcionando.  
Critical     -> Errores graves que impiden continuar la ejecución.
